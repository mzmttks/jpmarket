jpmarket
===============

Pandas DataReader extension for Japanese market data.

Pandas の DataReader で日本の市場データを読めるようにした拡張。


* Supported sources
  * yahoo ファイナンス
  * Ullet 企業情報
  * 

* Supported methods
  * DataReader("F", 'yahoo', start, end)

* Dependent packages
  * jsm
  * lxml
